# Greentic-X Contracts Reference Pack

This source pack packages the current Greentic-X reference contracts from the
repo-level `contracts/` tree into a `greentic-pack` contract scaffold.

Bundled assets:

- `assets/contracts/case`
- `assets/contracts/evidence`
- `assets/contracts/outcome`
- `assets/contracts/playbook`

The generated `contract-hook` component is only a placeholder scaffold so the
pack remains wizard-compatible. The useful payload is the mirrored contract
manifests, schemas, examples, and READMEs under `assets/contracts/`.
