# Greentic-X Ops Reference Pack

This source pack packages the current Greentic-X reference ops from the
repo-level `ops/` tree into a `greentic-pack` ops scaffold.

Bundled assets:

- `assets/ops/approval-basic`
- `assets/ops/playbook-select`
- `assets/ops/rca-basic`

The generated `ops-provider` component is a placeholder scaffold so the pack
shape stays wizard-compatible. The useful payload is the mirrored op manifests,
schemas, examples, and notes under `assets/ops/`.
